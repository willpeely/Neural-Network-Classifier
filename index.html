CNN Image Classification Model

This project implements a convolutional neural network (CNN) for image classification using PyTorch.

Repository Files

  • CNN.py - CNN architecture
  • train.py - Training script
  • test.py - Testing script
  • model.pth - Saved model weights
  • README.md - Project description

Results

Training Accuracy: 82.10%

Test Accuracy: 87.21%

How to Use

Clone the repository and run train.py to train the model, or test.py to test it using the saved model.pth.
